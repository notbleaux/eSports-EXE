# Changelog

All notable changes to the SATOR eXe project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-01

### Added
- **Landing Page**: Interactive SATOR Sphere with starfield background and warp transition
- **Launch Pad**: Mission control interface with constellation navigation
- **RadiantX Profile**: Valorant analytics dashboard with quaternary grid layout
  - Q1: Tactical Map with map selector
  - Q2: Observer View with live match feed
  - Q3: Data Rosarium with key statistics
  - Q4: Settings Ghost for configuration
- **Game Selector**: Dropdown for switching between game profiles (Valorant, CS, Apex)
- **Responsive Design**: Mobile-optimized layouts for all pages
- **Visual Effects**: 
  - Animated starfield background
  - Pulsing status indicators
  - Hover effects on interactive elements
  - ZN Junction morphing glyph animation

### Technical
- CSS custom properties for theming
- SVG-based SATOR Sphere with facet interactions
- Modular CSS architecture
- Keyboard shortcuts (1-5) for quick navigation

## [0.9.0] - 2026-02-28
