# Welcome to SATOR Documentation

**SATOR** (Strategic Analytics for Tactical Observation & Reporting) is a comprehensive esports analytics platform for Valorant and CS2.

## Quick Links

- [Contributing Guide](guides/contributing.md) - How to contribute
- [Deployment Guide](guides/deployment.md) - Deploy to production
- [API Reference](api/index.md) - API documentation
- [Troubleshooting](reference/troubleshooting.md) - Common issues

## Project Structure

```
apps/           - Applications (website, dashboard)
packages/       - Shared packages and libraries
services/       - Backend services
platform/       - Simulation platform
infrastructure/ - CI/CD and deployment
docs/           - Documentation
tests/          - Test suites
tools/          - Development tools
```

## Status

- **Website-v2**: React + Vite + Tailwind
- **API**: FastAPI + PostgreSQL + Redis
- **Simulation**: Godot 4 + C#
- **Grade**: B+ → A (in progress)

## Getting Started

```bash
git clone https://github.com/notbleaux/eSports-EXE.git
cd eSports-EXE
npm install
npm run dev
```
