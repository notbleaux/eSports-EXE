## 10. ANTI-REFERENCES (What NOT to do)

### Bootstrap Templates
**Avoid:** Generic admin dashboards with:
- Excessive rounded corners (16px+)
- Bootstrap blue (#007bff) as primary
- Card shadows everywhere
- Generic icon libraries

### Material Design
**Avoid:**
- Floating action buttons
- Ripple effects
- Excessive elevation shadows
- Pastel colors

### Dribbble "UI Kits"
**Avoid:**
- Overly stylized mockups
- Unrealistic data visualization
- Gradient-heavy designs
- Illustration-heavy layouts

## CAPTURE INSTRUCTIONS

When capturing screenshots:

1. **Browser:** Chrome at 1440px width
2. **Mode:** Dark mode enabled
3. **Zoom:** 100% (actual size)
4. **Format:** PNG with transparency if possible
5. **Annotation:** Highlight specific elements being referenced

### Tools for Capture:
- Chrome DevTools → Capture full-size screenshot
- Shottr (macOS) or ShareX (Windows)
