- **Integrations**: All hubs, World selector.

## 4. OPERA Tree (Action) - Flagship: Betting/Fantasy
- **Refined**: Odds/rewards calc, Reserve Bank bots, play tokens.
- **QoL**: Fantasy leagues auto-gen, bot watch mode, EV alerts.
- **Impl**: Real-time WebSocket, grafting AREPO data.
- **Integrations**: Betting sim, tokens circulation.

## 5. TENET Tree (Central Network) - Flagship: Modularity Hub
- **Refined**: Dial/cog (NJZ SFX toggle/glyphs), quarterly Tree ports.
- **QoL/Network**: Drag-drop grafting, connectivity diagnostics, World-Tree switcher, QoL search/export/share.
- **Impl**: Central Zustand, dynamic grafting (e.g., SATOR-OPERA-ROTAS combo), lensing views.
- **Interconnections**: Vines/physics/NJZ motifs link ports.

**Equitable Capacity**: Each flagship service balanced, NJZ-Base overlays all.
