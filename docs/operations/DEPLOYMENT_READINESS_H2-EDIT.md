## Deployment Steps

### 1. Supabase Database Setup
- Create free tier project at https://supabase.com
- Get connection string from Project Settings > Database
- Update `DATABASE_URL` in Render dashboard

### 2. Upstash Redis Setup
- Create free tier Redis at https://upstash.com
- Get Redis URL from console
- Update `REDIS_URL` in Render dashboard

### 3. Render Deployment
- Connect GitHub repo to Render
- Blueprint will auto-detect `render.yaml`
- Set required environment variables:
  - `PANDASCORE_API_KEY` (from pandascore.co)
  - `DATABASE_URL` (from Supabase)
  - `REDIS_URL` (from Upstash)

### 4. Run Migrations
```bash

./scripts/run-migrations.sh production
```

### 5. Verify Deployment
