About SATOR Square

The SATOR Square is a palindrome that reads the same forwards, backwards, and from any corner. Each letter represents a key analytical dimension:

  • S SimRating: Core performance rating combining 37 metrics
  • A Age Curve: Performance trajectory based on player age
  • T Temporal: Time-based analysis and trends
  • O Overfitting: Statistical validation metrics
  • R Role-Based: Position-adjusted performance
