# 4NJZ4 Heroes & Mascots Design Doc

## 5 Core Heroes (NJZ + ILLIT/LE SSERAFIM Hybrids - Unified Group)
Equitable, pop/hyperpop, fairy tale/Yahu Meha/Night Day/Romulus Remus themes.

1. **Sol (Leader/Day)**: NJZ Hanni primary; ILLIT Minju solo (dance grace), LE SSERAFIM Sakura hybrid (serene duality). Colours: Gold/Sunrise orange. Outfit: Jett cape hybrid. Synergy: Group speed boost.
2. **Lun (Support/Night)**: NJZ Danielle primary; LE SSERAFIM Chaewon solo (vocal harmony), ILLIT Yunah hybrid (gentle flow). Colours: Indigo/Moon silver. Outfit: Skye orbs. Synergy: Heal chain.
3. **Bin (Strategist/Twin Bind)**: NJZ Haerin primary; ILLIT Iroha solo (rhythm bind), LE SSERAFIM Kazuha hybrid (wind bind). Colours: Purple/Twin violet. Outfit: Astra stars. Synergy: Utility link.
4. **Fat (Fate Split)**: NJZ Hyein primary; LE SSERAFIM Yunjin solo (fate song), ILLIT Moka hybrid (split fate). Colours: Teal/Forked cyan. Outfit: Cypher wires. Synergy: Split defense.
5. **Uni (Overcomer/Friends)**: NJZ Minji primary; ILLIT Wonhee solo (unity beat), LE SSERAFIM Eunchae hybrid (friend power). Colours: Rainbow unity. Outfit: Breach blasts. Synergy: Team overdrive.
