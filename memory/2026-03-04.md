## Project Overview

Multi-modal eSports HUB interconnecting databases, dashboards, analytics services, and offline simulation games. Starting with Counter-Strike and Valorant, designed to scale to broader eSports ecosystem.

Built from strict budget constraints — free/low-cost infrastructure.

---

## Core Components

### 1. RAWS — Statistical Reference Website
**Inspiration:** Pro-Football-Reference.com, PFF Next Gen
- Static website with pro eSports statistical tables
- Reference-style presentation (NFL, NHL, MLB, Soccer, NBA, etc. as models)
- **Twin-table architecture:** RAWS tables paired with BASE class duplicates
- BASE extends from RAWS for advanced analytics but maintains parity
- Directory-managed integrity checks via central ping/verification system

### 2. Advanced Analytics Website (PFF-style)
- Parallel to RAWS with member/free tier differentiation
- **Free Users:** Basic grades and overall ratings only
- **Members:** Expanded statistical reference with derived/advanced/composite metrics
- Modern next-gen sports analytics methodologies
- Encrypted service, subscription-gated content

### 3. eXe Directory + NJZ Platform
- **eXe:** Central directory for all project systems
- **NJZ:** Tools platform providing:
  - Website tools and dashboards
  - Developer & user dashboards
  - Health/maintenance monitoring
  - Formula glossary and database management
  - Tools for each segment/section of the ecosystem

### 4. AXIOM eSports — Offline Video Game
**Genre:** 2D Strategy/Simulation Management  
**Inspiration:** Football Manager series, Crusader Kings, Sim-style games
- Offline eSports manager simulation
- Integrates with analytics databases for realistic modeling

### 5. NJZ Stock-Trading Simulation
**Purpose:** Economic modeling & formula validation engine
- Paper stock trading system redesigned for eSports market simulation
- Real-time economic indexing of:
  - Analysis metrics
  - Win% predictions
  - Power rankings
  - Betting market values
- **Function:** Validates prediction efficacy of analytics metrics
- Autonomous daily evaluations and reports
- Feeds back into formula fine-tuning for other systems
- **Status:** Very early concept/prototype wanted
