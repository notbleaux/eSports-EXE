## SATOR-eXe-ROTAS

A complex, interconnected eSports ecosystem starting with CS/Valorant. Built on strict budget constraints.

**Core Philosophy:** Twin-table data integrity (RAWS/BASE), central directory management, parity-checked interconnected systems.

### Components:
1. **RAWS** — Static stats reference site (Pro-Football-Reference style)
2. **Advanced Analytics** — PFF-style member site with derived metrics
3. **eXe/NJZ** — Central directory and tools platform
4. **AXIOM eSports** — Offline 2D management simulation game
5. **NJZ Market Sim** — Stock-trading style formula validation engine

**Current Phase:** Early concept/prototype

- **Aesthetic:** Functional, sports-reference inspired (not AI-slop gradients)
