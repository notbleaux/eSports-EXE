## 🏛️ Overview

This is the **Gilded Legacy Repository** for the satorXrotas platform — a comprehensive esports analytics system built on the twin-file philosophy of **RAWS** (Raw Archive Write System) and **BASE** (Base Analytics System for Esports).

The repository follows strict versioning standards with `[VerMMM.mmm]` headers on all files, enabling clear lineage tracking from the original 2024 vanilla JavaScript implementation through the current React 18 architecture.

## 🎨 The SATOR Philosophy

The platform embodies the ancient **SATOR Square** — a 5×5 palindrome that reads the same forwards, backwards, and in reverse:

```
S A T O R
A R E P O
T E N E T
O P E R A
R O T A S
```

**SATOR** = The sower (data ingestion)  
**ROTAS** = The wheels (analytics processing)

This recursive, self-referential structure mirrors our twin-file architecture:
- **RAWS** receives raw data (SATOR)
- **BASE** processes and transforms (ROTAS)
