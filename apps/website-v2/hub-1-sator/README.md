# SATOR Hub - The Observatory

Abyssal depth data visualization hub with signal-amber accents for the NJZ Platform.

## Features

- **5 Concentric Orbital Rings** - CSS 3D transformed animated rings representing Teams, Matches, Players, Tournaments, and History
- **Particle Star Field** - Three.js particle system with 1500+ orbiting data points
- **Lissajous Comparator** - Canvas-based harmonic curves for dataset comparison
- **Terminal Verifier** - ASCII terminal aesthetic with SHA-256 integrity checks
- **Materiality Toggle** - Observable/Scalar spectrum slider based on Dale Pond's concepts
- **60fps Optimized** - Smooth animations with requestAnimationFrame
- **Responsive Design** - Adapts to all screen sizes
- **Smoke-like Transitions** - Smooth state transitions

## Visual Design

- **Abyssal Depth Aesthetic** - Deep void blacks with subtle amber accents
- **Signal Amber Primary** - #ff9f1c as the main accent color
- **Phosphor Glow Effects** - Terminal-style amber glow
- **Scan Lines Overlay** - Subtle CRT-style effect
- **Ambient Particles** - Floating data points in background

Open `index.html` in a modern browser, or serve with:
