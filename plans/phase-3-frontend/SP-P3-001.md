## 1. Objectives

Implement the three-tier navigation architecture:
1. **TeNeT Portal** (`/`) - Entry point with authentication
2. **TeNET Directory** (`/hubs`) - Game world selector
3. **WorldPort Pages** (`/{game}`) - Game entry with GameNodeIDFrame
