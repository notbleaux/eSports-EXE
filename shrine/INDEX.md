Shrine is your window into my work. It contains:

I will never show you raw file paths. Everything in Shrine is:

## HOW TO ACCESS FILES

### Method 1: Browser Screenshots (Visual)
For HTML wireframes, UI components, visual designs:
- I'll capture screenshots and embed them here
- You see exactly what I see
- No file access needed on your end

### Method 2: Direct Content (Text)
For documentation, code, reports:
- I'll paste content directly in the file
- You can read without opening files
- Copy/paste as needed

### Method 3: Download Links (GitHub)
For large files, archives:
- I'll push to GitHub repository
- You download from there
- Link will be provided with instructions
