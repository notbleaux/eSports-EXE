# 01_LISTINGS
## Available Job Listings

**Purpose:** Active and archived job listings for agent assignment

---

## Structure

### ACTIVE/
Current job listings available for claiming.

Files follow naming convention:
`
{CATEGORY}-W{WAVE}-{AGENT}-{DESCRIPTION}.md
`

Examples:
- HELP-W1_1-1A_CONTENT_SCHEMA.md
- SPEC-W1_1-1B_ANALYTICAL_LENSES.md

### ARCHIVED/
Historical job listings (completed or obsolete).

---

## How to Use

1. Browse ACTIVE/ for available tasks
2. Review job description
3. Claim job following JOB_CLAIMING_PROTOCOL_v2.md
4. Move to 02_CLAIMED/ upon claiming
